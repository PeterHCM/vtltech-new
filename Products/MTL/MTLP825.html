Design Features

  • DIN size module with 8, 16 and 24 channel versions
  • Internally generated 24VDC signal supply, with options for powered inputs 24VDC, 125VAC/DC, 48VDC or 250VAC/DC
  • Pluggable LEDs in five colors: Red, Yellow, White, Blue and Green
  • Remote pushbutton inputs with mappable functions
  • Each channel fully software configurable via front panel USB port using intuitive setup utility
  • Integrated power supply for direct connection to 85–264VAC or 88–300VDC
  • Optional wide-range DC supply: 19–72VDC
  • Optional internal redundant power supply
  • Full ISA 18.1 alarm sequences built-in, programmable via front panel USB connection
  • Flush panel units rated IP40, with optional IP54 kit
  • No links required for relay configuration
